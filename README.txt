This assigmets is a simple implementation of the factory desing patter.
We took the original implemenation from the video by Derek Banas : https://youtu.be/ub0DXaeV6hA?si=uCCLkxkBDTkU3x9U 
and extended it to incorporate two new classes: Super destroyer and space pirate.

To run this code :
1) clone the repo and navegate to teh directory
2) compile the testing file wiht : ❯ javac EnemyShipTesting.java
3) run the file wiht : ❯ java EnemyShipTesting
4) input one of the options promted to see the factory get called and build the chosen ship.
